CS141 Final Project
Nicholas Novak

For this project, I created a JavaFX program which generates and solves an electrical circuit according to a randomized template while hiding the answers from user. The user can utilize their knowledge of circuit behavior to determine the unknown values, and check to see if their answers are correct. The elements of this circuit include either a voltage source or a current source, and parallel paths of resistors. A couple switches in the circuit allow for some randomization, so that each circuit will be unique.

The UI shows a depiction of the circuit with labels on each component that update with new values for each newly generated circuit, and switches which open or close which alter the path. The user will fill out a table of values for the voltage and current across each resistor and then press a 'Check Answers' button that outlines incorrect answers in red and correct answers in green. There is also another button to generate a new circuit, which will randomize the switches and component values.
